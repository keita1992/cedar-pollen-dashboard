# 🌲 東京都スギ花粉ダッシュボード

東京都の12観測地点におけるスギ花粉飛散データを可視化するダッシュボードです。

## 掲載データ

- **対象年度**: 2019年（平成31年）、2021年（令和3年）〜2025年（令和7年）
- **観測地点**: 千代田、葛飾、杉並、北、大田、青梅、八王子、多摩、町田、立川、府中、小平
- **単位**: 個/cm²

## 機能

- 📈 日別花粉飛散量の推移（折れ線グラフ）
- 🔄 年度別比較チャート
- 📊 累積飛散量グラフ
- 🏙️ 地点別シーズン合計（横棒グラフ）
- 📅 月別飛散量
- 🏆 ピーク日ランキング
- 年度・地点の切り替えフィルター

## データ出典

東京都福祉保健局「花粉情報」アーカイブ
